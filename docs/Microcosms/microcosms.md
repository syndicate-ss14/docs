# Microcosms

A Microcosm is a voting member community of Macrocosm.

### Becoming a Microcosm

Accepting a new Microcosm requires unanimous agreement on part of all current Microcosms. You may express your interest in becoming a Microcosm by opening a GitHub issue here. <!-- TODO make github issue template for membership request and then link it here. after macro repo is set up -->

You do not have to be part of a Microcosm to use Macrocosm, or to contribute to Macrocosm.
